# Holiday Calender

Shows upcoming holidays from countries around the world.

## Usage

```sh
python next_holidays.py [-n COUNT]
```

- `-n COUNT` — number of upcoming holidays to display (default: 20)

### Examples

```sh
python next_holidays.py
python next_holidays.py -n 5
python next_holidays.py -n 50
```

## Data

Powered by the [holidays](https://github.com/vacanza/python-holidays) library, covering 50+ countries and many subdivisional regions.
